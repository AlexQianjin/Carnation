# Carnation

# How to run
## client
- install dependencies and run the app:
```
cd client/
$ npm install 
$ npm start
```

## commonpage
- install dependencies and run the app:
```
cd commonpage/
$ npm install 
$ npm start
```

## server
- install dependencies and run the app:
```
cd server/
$ npm install 
$ npm start
```
